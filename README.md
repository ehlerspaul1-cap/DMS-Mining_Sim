# DMS Mining Simulator V2

This is the staged rebuild of the DMS Mining educational simulator.

## Start locally

Open `popup-sim.html` in Chrome or Edge, then select **Start simulator**.

An internet connection is required for the AnyChart and seeded-random libraries.

## First release

- Responsive landing page and simulator interface.
- Seven guided objectives from geophysics to first production.
- Corrected project budget, copper value and drilling calculations.
- Working graph show/hide controls.
- Priced people and exploration-rig upgrades.
- Responsive mining canvas with corrected click coordinates.

## Staging deployment

1. Create `/public_html/sim-v2/` in cPanel File Manager.
2. Upload the contents of this folder or the staging ZIP.
3. Test `https://dms-mining.com.na/sim-v2/popup-sim.html`.
4. Leave the existing `/public_html/sim/` folder unchanged until approval.

See `ROADMAP.md` for the proposed incremental releases.
